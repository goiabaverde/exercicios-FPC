# exercicios-FPC

Neste repositório estão os códigos desenvolvidos durante a disciplina **Fundamentos da Programação de Computadores**.

## Sobre o repositório

O repositório contém os exercícios e atividades realizados ao longo da disciplina **Fundamentos da Programação de Computadores**, que faz parte do curso de **Bacharelado em Física** da **USP – campus São Carlos**.

Os códigos foram escritos na linguagem **Python**, e os problemas resolvidos durante a disciplina abrangem desde a simulação de sistemas físicos até algoritmos de ordenação e busca, além de tópicos de cálculo numérico.
